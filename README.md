## Getting Started

These instructions will help you get a copy of the project up and running on your local machine.

### Prerequisites

- .NET Core SDK


### Installing

1. Clone the repository to your local machine.

```bash
git clone https://github.com/ThBraum/BinksNoSake.git
cd your-repository
```


2. Run.
```console
dotnet build
dotnet run
```
